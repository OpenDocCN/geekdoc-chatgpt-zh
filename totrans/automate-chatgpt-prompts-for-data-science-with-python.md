7. 训练时间序列

提示：我想让你扮演数据科学家并为我编写代码。我有一个时间序列数据集[描述数据集]。请构建一个机器学习模型来预测[target 变量]。请使用[时间范围]作为训练集，[时间范围]作为验证集。

![image_rsrc10B](img/automate-chatgpt-prompts-for-data-science-with-python-image_rsrc10B.jpg)![image_rsrc10C](img/automate-chatgpt-prompts-for-data-science-with-python-image_rsrc10C.jpg)![image_rsrc10D](img/automate-chatgpt-prompts-for-data-science-with-python-image_rsrc10D.jpg)![image_rsrc10E](img/automate-chatgpt-prompts-for-data-science-with-python-image_rsrc10E.jpg)

ChatGPT 提供了以下代码：

+   使用标准缩放预处理数据

+   创建滞后变量

+   训练 RandomForestRegressor 进行评估

+   使用平均绝对误差进行评估

好吧，这是一个有争议的问题。ChatGPT 无法正确编写时间序列验证的代码。

原因如下：没有提示“请使用 A 到 B 的数据作为训练。请使用 Y 到 Z 的数据作为验证”，ChatGPT 使用 sklearn 的 train_test_split 随机分割数据点为训练集和验证集。这是一个大忌，因为 train_test_split 函数随机分割数据时，不考虑数据集之间的时间关系。相反，它应该使用早期时间框架的所有数据点作为训练集，后期时间框架作为测试集。

8. 解决数据不平衡问题

提示：我想让你扮演编码者。我在一个不平衡的数据集上训练了一个机器学习模型。预测变量是[插入列名]列。在 Python 中，我如何进行过采样和/或欠采样我的数据？

![](img/automate-chatgpt-prompts-for-data-science-with-python-image_rsrc10F.jpg)![](img/automate-chatgpt-prompts-for-data-science-with-python-image_rsrc10G.jpg)

ChatGPT 使用了 imblearn 库来编写样板代码，该代码随机对数据集进行过采样和欠采样。

代码是合理的，但我对其对过采样和欠采样的理解有些挑剔。

过采样和欠采样只应在训练数据集上进行。它不应在整个数据集上进行，包括测试数据集。否则，评估将在不再反映真实数据集实际分布的重新采样测试集上进行。

因此，它应该首先尝试将数据集拆分为训练集和测试集，然后再进行过采样和欠采样。
